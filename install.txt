Чтобы не запутаться последовательность действий такая:
1. Установить github desktop
2. Склонировать проект alien_invasion себе на комп
3. Установить pygame из консоли анаконды или из навигатора (команды см. ниже)
4. После открытия проекта alien_invasion и запуска должна запускаться игра как я сегодня показывал
5. Можно начинать вносить правки в код под наши нужды
